# Skills

AI skills that enhance DevOps capabilities.

## Available Skills

### security-review
Analyzes infrastructure code for security vulnerabilities, compliance issues, and provides hardening recommendations.

- Terraform security analysis
- IAM policy review
- Network security
- Secrets management
- Compliance mapping (SOC2, PCI, HIPAA)

### finops-aws
Analyzes cloud infrastructure costs, reviews Terraform configurations, and provides cost optimization recommendations.

- Cost analysis
- Right-sizing recommendations
- Waste identification
- Pricing optimization

## Usage

These skills are automatically invoked by the `devops-entry` agent when you request:
- Security reviews: "security", "review", "audit", "vulnerability"
- Cost reviews: "cost", "finops", "pricing", "optimize"
